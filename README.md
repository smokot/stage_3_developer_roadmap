# stage_3_developer_roadmap
* HTML5 (семантическая вёрстка)

Базовая структура: `<!DOCTYPE html>, <head>, <meta>, <title>, <body>.

Семантические теги: <header>, <nav>, <main>, <article>, <section>, <aside>, <footer>.

Текстовые: <h1>–<h6>, <p>, <ul>/<ol>/<li>, <a>, <strong>/<em>.

Формы: <form>, <input> (все типы: text, email, password, number, date, file), <textarea>, <select>, <label>, <button>, атрибуты required, placeholder, name (критично для PHP!).

Таблицы: <table>, <thead>, <tbody>, <tr>, <th>, <td>.

Мультимедиа: <img>, <video>, <audio>, атрибут alt для доступности.

Атрибуты id, class, data-*.

Понятие валидного HTML и валидатор W3C.`
